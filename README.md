# Frontend Mentor - Article preview component

## Welcome! 👋🏻

Thank you for checking out my attempt at this front-end mentor coding challenge. My goal for this challenge was to create an article preview component that's both web and mobile responsive. The objective was to get a close to the design provided as possible.

[Article preview component challenge on Frontend Mentor](https://www.frontendmentor.io/challenges/article-preview-component-dYBN_pYFT).
